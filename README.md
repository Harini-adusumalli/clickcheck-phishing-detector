# ClickCheck - Phishing Detection System

## Project Overview
ClickCheck is an AI-based phishing detection system designed to identify malicious URLs using structural analysis, machine learning, and contextual similarity.

## Feature Format (IMPORTANT)
All modules must use this format:

[length, dots, https]

Example:
https://google.com → [18, 1, 1]

## Folder Structure
backend/ → Feature extraction & FastAPI  
ml_model/ → ML model  
vector_db/ → ChromaDB setup  
frontend/ → UI  
docs/ → documentation  

## Team Workflow
Each member works independently but follows the same feature format.
